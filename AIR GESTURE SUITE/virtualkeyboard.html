Virtual
Keyboard

The Virtual Keyboard application offers an intuitive and hands-free way to type on a virtual keyboard using hand gestures. This system is particularly useful in environments where touch-based interactions may be inconvenient. Utilizing a webcam, the application tracks hand gestures to perform typing actions on an active text window like Notepad. With the integration of color detection to enhance gesture tracking, especially with green objects, and advanced features like font size, font color, and other text-related settings, the application provides a full-fledged virtual typing experience which we plan to implement in stage-3.
